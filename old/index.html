What is BENOG?

BENOG stands For 'Belgian Network Operators Group', and is a forum for the Belgian Network Operators

What is BENOG offering?

The BENOG server facilitates a Mailinglist,wiki and a chat which can be publicly and freely used by
Belgian Network Operators
