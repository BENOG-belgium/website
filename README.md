# beNog Website :
## What :
This is the brand new website of the beNOG. Old one was only static HTML. This one has been planned to be able to plan events.

## Run :
```
virtualenv ve3
source ve3/bin/activate
pip install -r requirements.txt
./manage.py migrate
./manage.py runserver
```
